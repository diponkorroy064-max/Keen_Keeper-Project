## :house: Home Page 
Presenting a Hero section with Title, Sub-title and a button, showing 4 summsry Card bellow the Hero  or Bannar Section. Bellow the summary card 12 friends cards are presented by dinamically by fetching using global hooks. It is also a 4 grid layout.

## :timer_clock: Timeline Page
This page shows the history of all interactions (calls, texts, video calls) logged from the Friend Details page.
This page must have:-
#### :gear: A "Timeline" heading at the top
#### :gear: Timeline entries displayed following-
#### :gear: Each entry must show:
#### :gear: Field Details-

#### 📅 Date:-  
When the interaction happened

#### 🔣 Icon:-
Different icon for Call / Text / Video

#### 📝 Title:-
e.g., "Call with Sarah", "Text with Mike"


## :bar_chart: Stats Page 
It show a beautiful Pie Chart. The Pie Chart present interactions of call, text, video, meetup dynamically. The data of the Pie Chart comes from the Context API.


## :person_in_tuxedo: Friend Details Page
It shows the all information about my friends by dynamically from JSON data. It presents the 2 grid column. In the left side column It present a Friends Card coming information dynamically. Below the friends card showing 3 buttons. 
In the right side column it present 3 sections:-

#### 1. Stats Cards (show 3 cards):-
1. Days Since Contact
2. Goal (in days)
3. Next Due Date

#### 2. Relationship Goal Card:-
It show the current contact goal and Addition of an Edit button.

#### 3. Quick Check-In Card:-
Addition of 3 buttons with icons: Call, Text, Video
When clicked, each button adds a new entry to the Timeline (see Section 6).

#### 4. Recent Interactions Cards:-
Shows the interactions about call, text, video and meetup.


## :mag_right: Footer Section:-
It represents footer title, subtitle and some socisl links.
